Welcome 2019 SfN Antendees to DataJoint's Public
Jupyter environment! Here you may explore what is
capable with DataJoint Python via Jupyter Notebooks.
All that is needed is a GitHub login. This Jupyter
environment showcases the DataJoint canonical
pipelines for electrophysiology and behavior, with
data from published studies. For reference, we have
included some sample notebooks within the shared
directory in the root. However, keep in mind that
only items in your HOME will be persisted i.e. the
directory named after your GitHub ID. Therefore, if
pip install is necessary, make sure to utilize the
--user flag. Feel free to join the community in our
DataJoint Slack so you can let us know what you think!
Happy DataJoint'ing!!
